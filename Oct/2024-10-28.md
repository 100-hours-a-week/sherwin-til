#### Git: Git 명령어 기본
- git init       : 작업 공간을 생성한다
- git remote add : 원격 저장소명을 지정하고 생성한다
                   -v 옵션을 사용해서 저장소 조회가 가능하다
- git add        : 파일을 commit 하기위해 stage한다
- git commit     : 파일의 변경 내용을 확정한다
