### 새로 배운 내용
#### HTTP
- HTTP : 클라이언트와 서버 사이에 이루어지는 요청/응답 프로토콜이다
         웹어플리케이션을 실행하게 되면 url을 통해 클라이언트가 도메인에 HTTP를 보내게 된다

#### HTTP - Request Method
- HTTP메서드 : 서버가 수행해야 할 동작을 지정하여 요청을 보내는 방법이다
               GET, POST, PUT, DELETE 등이 있다                  

#### HTTP - Status Code
- 상태코드 :  HTTP 요청이 성공적으로 완료되었는지 알려준다
